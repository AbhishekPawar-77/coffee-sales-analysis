☕ Coffee Shop Sales Analysis (Excel Dashboard Project)

📌 Project Overview

This project presents an interactive Coffee Shop Sales Dashboard built using Microsoft Excel.
It helps analyze sales performance, customer behavior, and product trends.

---

📊 Tools & Techniques Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning

---

📈 Dashboard Features

- 📅 Month-wise & Day-wise Filters
- ⏰ Sales by Hour Analysis
- 🛍️ Category-wise Sales Distribution
- 📦 Top 5 Products Based on Sales
- 🏪 Store Location Performance
- 📊 Weekday Order Trends

---

🔢 Key Metrics

- 💰 Total Sales: $698,812.33
- 👥 Total Footfall: 149,116
- 🧾 Avg Bill/Person: 4.69
- 🛒 Avg Orders/Person: 1.44
