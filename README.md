☕️ Coffee Shop Sales Analysis (Excel Dashboard Project)
📌 Project Overview
This project presents an interactive Coffee Shop Sales Dashboard built using Microsoft Excel.
It helps analyze sales performance, customer behavior, and product trends across different time periods and store locations.
📊 Tools & Techniques Used
Microsoft Excel
Pivot Tables
Pivot Charts
Data Cleaning and Transformation
Dashboard Design 
📈 Dashboard Features
📅 Month-wise & Day-wise Filters for dynamic analysis
⏰ Sales by Hour Analysis to identify peak business hours
🛍️ Category-wise Sales Distribution
📦 Top 5 Products Based on Sales
🏪 Store Location Performance (Sales & Footfall)
📊 Weekday Order Trends
📐 Order Size Distribution Analysis
🔢 Key Metrics
💰 Total Sales: $698,812.33
👥 Total Footfall: 149,116
🧾 Average Bill per Person: 4.69
🛒 Average Orders per Person: 1.44
🔍 Key Insights
Peak order quantity is observed during morning to midday hours
Certain product categories contribute significantly to total revenue
Top 5 products generate a major portion of total sales
Sales performance varies across different store locations
Weekdays show consistent ordering trends with slight variations
